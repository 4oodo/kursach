# Администраторское приложение управления энергопотреблением

Это приложение предназначено для администраторов системы управления энергопотреблением в зданиях.

## Функциональность

### 1. Управление зданиями
- Просмотр всех зданий
- Добавление новых зданий
- Редактирование информации о зданиях
- Удаление зданий

### 2. Управление категориями помещений
- Просмотр всех категорий (Офис, Коридор, Склад и т.д.)
- Добавление новых категорий
- Редактирование категорий
- Удаление категорий

### 3. Управление периодами времени
- Просмотр всех периодов времени
- Добавление новых периодов
- Редактирование периодов
- Удаление периодов

### 4. Управление помещениями
- Просмотр всех помещений с информацией о зданиях и категориях
- Добавление новых помещений
- Редактирование данных помещений
- Удаление помещений

### 5. Управление энергопотреблением
- Просмотр всех данных энергопотребления
- Добавление новых записей потребления энергии
- Редактирование записей
- Удаление записей
- Фильтрация данных по помещениям и датам

### 6. SQL консоль
- Выполнение произвольных SQL запросов
- Просмотр результатов в табличном виде
- Поддержка SELECT, INSERT, UPDATE, DELETE операций

### 7. Формирование отчетов

#### Отчет об энергопотреблении
- Генерирование отчетов за выбранный диапазон дат
- Информация о энергопотреблении по зданиям, помещениям и периодам времени
- Расчет кумулятивного потребления

#### Статистика по зданиям
- Общая статистика по зданиям
- Количество помещений в каждом здании
- Общее, среднее, максимальное и минимальное энергопотребление

## Инструкция по установке и запуску

### Требования
- .NET Framework 4.7.2
- SQL Server 2017 или новее
- Visual Studio 2019 или нове

### Шаг 1: Подготовка базы данных

1. Откройте SQL Server Management Studio
2. Выполните SQL скрипт для создания БД:
   ```sql
   CREATE DATABASE EnergyManagement ON PRIMARY ( 
	   NAME = N'EnergyManagement', 
	   FILENAME = N'E:\Microsoft sql\MSSQL17.MSSQLSERVER\MSSQL\DATA\EnergyManagement.mdf', 
	   SIZE = 10MB, 
	   FILEGROWTH = 10% 
   ) LOG ON ( 
	   NAME = N'EnergyManagement_log', 
	   FILENAME = N'E:\Microsoft sql\MSSQL17.MSSQLSERVER\MSSQL\DATA\EnergyManagement_log.ldf', 
	   SIZE = 5MB, 
	   FILEGROWTH = 10% 
   );
   ```

3. Создайте таблицы (выполните соответствующий SQL скрипт из файла инициализации)

### Шаг 2: Конфигурация приложения

1. Откройте `App.config` и проверьте строку подключения:
   ```
   Data Source=DESKTOP-N513RVN;Initial Catalog=EnergyManagement;Integrated Security=True;
   ```
   Измените на вашу конфигурацию если необходимо.

2. Аналогично обновите строку подключения в `Services\DatabaseService.cs` в конструкторе класса

### Шаг 3: Запуск приложения

1. Откройте проект в Visual Studio
2. Нажмите F5 или выберите Debug -> Start Debugging
3. Приложение откроется с главным окном администратора

## Использование вкладок

### Вкладка "Управление зданиями"
- Введите название и адрес здания
- Нажмите "Добавить"
- Для редактирования дважды кликните на строку в таблице
- Введите новые данные и нажмите "Обновить"
- Выберите здание и нажмите "Удалить" для удаления

### Вкладка "SQL консоль"
- Введите SQL запрос в текстовое поле
- Нажмите "Выполнить запрос"
- Результаты отобразятся в таблице ниже
- Используйте "Очистить" для очистки запроса

### Вкладка "Отчеты"
- Отчет об энергопотреблении: выберите даты и нажмите "Генерировать отчет"
- Статистика по зданиям: нажмите "Генерировать статистику"

## Примеры SQL запросов

```sql
-- Получить все энергопотребление за дату
SELECT * FROM dbo.EnergyConsumption WHERE [Date] = '2024-01-01'

-- Получить сумму потребления по зданиям
SELECT b.BuildingName, SUM(ec.EnergyValue) as TotalEnergy
FROM dbo.EnergyConsumption ec
JOIN dbo.Room r ON ec.RoomID = r.RoomID
JOIN dbo.Building b ON r.BuildingID = b.BuildingID
GROUP BY b.BuildingName

-- Получить помещения с наибольшим потреблением
SELECT TOP 10 r.RoomID, b.BuildingName, r.Floor, SUM(ec.EnergyValue) as TotalEnergy
FROM dbo.EnergyConsumption ec
JOIN dbo.Room r ON ec.RoomID = r.RoomID
JOIN dbo.Building b ON r.BuildingID = b.BuildingID
GROUP BY r.RoomID, b.BuildingName, r.Floor
ORDER BY TotalEnergy DESC
```
