# 🔍 СПРАВОЧНИК ПО ФИЛЬТРАЦИИ И ПОИСКУ ДАННЫХ

## 📊 Полный список фильтров в приложении

---

## 🏢 Фильтры по зданию (Building)

### Где используется:
- ✅ Вкладка "Помещения" - фильтр для отбора помещений по зданиям
- ✅ Вкладка "Энергопотребление" - фильтр для отбора энергопотребления по зданиям
- ✅ Вкладка "Поиск" - критерий поиска по зданиям

### Как работает:
```
Выбираете здание → Система находит все помещения этого здания → 
Отображает данные только для этого здания
```

### Пример:
- Выбрали "Корпус А" → видим только помещения в Корпусе А
- Выбрали "Главное здание" → видим только помещения в Главном здании

---

## 🚪 Фильтры по помещению (Room)

### Где используется:
- ✅ Вкладка "Энергопотребление" - фильтр для отбора данных по конкретному помещению

### Как работает:
```
Выбираете помещение → Система показывает энергопотребление только этого помещения
```

### Особенности:
- ⚠️ Список помещений заполняется **только после выбора здания**
- 💡 После выбора здания можно выбрать любое помещение из этого здания

### Пример:
1. Выберите здание "Главное здание"
2. В комбо-боксе "Помещение" появятся: Помещение 1, Помещение 2, ...
3. Выберите конкретное помещение
4. Таблица обновится с энергопотреблением этого помещения

---

## 📅 Фильтры по дате (Date)

### Где используется:
- ✅ Вкладка "Поиск" - критерии "От даты" и "До даты"

### Как работает:
```
От даты: начало периода (включительно)
До даты: конец периода (включительно)

Система находит все записи: От даты ≤ Дата записи ≤ До даты
```

### Примеры:
| От даты | До даты | Результат |
|---------|---------|-----------|
| 01.01.2024 | 31.01.2024 | Данные за январь 2024 |
| 01.06.2024 | 30.06.2024 | Данные за июнь 2024 |
| 01.01.2024 | 31.12.2024 | Данные за весь 2024 год |

### Примечание:
- 📌 Если только выбрали "От даты" - будут показаны данные **после** этой даты (включительно)
- 📌 Если только выбрали "До даты" - будут показаны данные **до** этой даты (включительно)
- 📌 Если выбрали обе даты - будут показаны данные **в диапазоне** (включительно с обеих сторон)

## ⏰ Фильтры по времени суток (Time Period)

### Где используется:
- ✅ Вкладка "Энергопотребление" - фильтр по периодам времени

### Стандартные периоды:
- 🌙 **Ночь** - обычно 00:00 - 08:00
- ☀️ **День** - обычно 08:00 - 20:00
- 🌙 **Вечер** - обычно 20:00 - 24:00

### Как работает:
```
Выбираете период (например, "День") → Система показывает энергопотребление 
только в дневное время для выбранного помещения
```

### Пример:
1. Выберите здание
2. Выберите помещение
3. Выберите период "День"
4. Увидите только дневное энергопотребление этого помещения

## 🔄 Комбинирование фильтров

### На вкладке "Энергопотребление"

Все фильтры работают одновременно (логика И):

```
Здание = "Корпус А" И 
Помещение = "Помещение 5" И 
Период = "День"
↓
Результат: Дневное энергопотребление помещения 5 в Корпусе А
```

### На вкладке "Поиск"

Фильтры также комбинируются:

```
Здание = "Корпус А" И 
От даты = "01.06.2024" И 
До даты = "30.06.2024"
↓
Результат: Все энергопотребление в Корпусе А за июнь 2024
```

| Вкладка | Здание | Помещение | Дата | Период | Заметки |
|---------|--------|-----------|------|--------|---------|

**Легенда:**
- ✅ = Фильтр доступен
- ❌ = Фильтр недоступен
- \* = Особое условие

## 🎯 Сценарии использования

### Сценарий 1: Проверить энергопотребление одного помещения

1. Откройте вкладку **"Энергопотребление"**
2. Выберите здание в комбо-боксе "Здание"
3. Выберите помещение в комбо-боксе "Помещение"
4. (Опционально) Выберите период
5. ✅ Готово! Видите энергопотребление

### Сценарий 2: Сравнить потребление по периодам

1. Откройте вкладку **"Энергопотребление"**
2. Выберите помещение
3. Видите все периоды для этого помещения
4. Переключайте период, чтобы сравнить

### Сценарий 3: Найти данные за конкретный месяц

1. Откройте вкладку **"Поиск"**
2. Нажмите на "От даты" и выберите 1-е число месяца
3. Нажмите на "До даты" и выберите последнее число месяца
4. (Опционально) Выберите здание
5. Нажмите "Искать"
6. ✅ Получите все данные за этот месяц

### Сценарий 4: Получить статистику по зданиям

1. Откройте вкладку **"Отчеты"**
2. Выберите "Общее энергопотребление по зданиям"
3. Нажмите "Сформировать отчет"
4. ✅ Видите статистику по всем зданиям

## ⚡ Оптимизация при работе с большими объемами данных

### Совет 1: Используйте фильтры
- Большой объем данных? → Примените фильтры
- Нужны данные за год? → Разбейте на месячные периоды

### Совет 2: Начните с помещения
- Выберите конкретное помещение перед выбором периода
- Это ускорит загрузку и фильтрацию

### Совет 3: Используйте поиск
- Если нужна точная дата → используйте вкладку "Поиск"
- Она оптимизирована для поиска по диапазону дат

### Совет 4: Проверяйте статистику в отчетах
- Вместо просмотра всех строк → используйте готовые отчеты
- Они показывают агрегированные данные (сумму, среднее, максимум)

## 🔔 Важные замечания

✅ **Все фильтры работают в реальном времени**
- При изменении фильтра таблица обновляется автоматически
- Не нужно нажимать кнопку "Применить"

✅ **Комбинирование фильтров**
- Все выбранные фильтры применяются одновременно
- Это логика И (AND), не ИЛИ (OR)

✅ **Сброс фильтров**
- В "Помещениях" есть кнопка "Показать все"
- На других вкладках просто отмените выбор в комбо-боксе

✅ **Чувствительность к регистру**
- Фильтры НЕ чувствительны к регистру (A = a)
- Пробелы учитываются
