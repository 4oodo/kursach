# Личный кабинет пользователя - Руководство пользователя

## Вход в систему

1. **Запустите приложение** - откроется окно входа
2. **Введите учетные данные:**
   - Администратор: `admin` / `admin123`
   - Пользователь 1: `user1` / `user123`
   - Пользователь 2: `user2` / `user123`
3. **Нажмите "Войти"** или **"Зарегистрироваться"** для создания нового аккаунта

## Личный кабинет пользователя

После входа пользователь получает доступ к 5 вкладкам:

### 1. 📦 Вкладка "Здания"
- Просмотр списка всех зданий в организации
- Информация: ID, название, адрес
- **Возможности пользователя:** только просмотр (без редактирования)

### 2. 🏢 Вкладка "Помещения"
- Просмотр всех помещений с возможностью фильтрации по зданиям
- Информация: ID помещения, этаж, категория, здание
- **Фильтр по зданию:** выберите здание из выпадающего списка
- **Кнопка "Показать все":** отобразить помещения всех зданий

### 3. ⚡ Вкладка "Энергопотребление"
- Просмотр данных потребления электроэнергии
- **Фильтры:**
  - По зданию
  - По помещению (автоматически обновляется при выборе здания)
  - По периоду времени (Ночь, Утро, День, Вечер)
- Таблица показывает: дату, время, помещение, значение потребления

### 4. 🔍 Вкладка "Поиск"
- Расширенный поиск по данным энергопотребления
- **Критерии поиска:**
  - Здание (обязательно)
  - Дата "от" (опционально)
  - Дата "до" (опционально)
- **Кнопка "Искать":** выполнить поиск с выбранными параметрами
- Результаты отображаются в таблице

### 5. 📊 Вкладка "Отчеты"
- Генерирование аналитических отчетов
- **Типы отчетов:**
  1. Общее энергопотребление по зданиям
  2. Энергопотребление по помещениям
  3. Энергопотребление по периодам
- **Кнопка "Сформировать отчет":** создать выбранный отчет

## Кнопка "Выход"

- Расположена в правом верхнем углу
- При нажатии вернет вас на экран входа

## Тестовые данные

В БД уже добавлены тестовые данные:
- **3 здания:** Главное здание, Корпус А, Корпус Б
- **6 категорий помещений:** Офис, Коридор, Склад, Кафе, Серверная, Лестничная клетка
- **4 временных периода:** Ночь, Утро, День, Вечер
- **20+ помещений** распределены по зданиям
- **30+ записей** энергопотребления

## Администратор

Администраторы получают полный доступ:
- Управление всеми сущностями (CRUD операции)
- SQL консоль для выполнения запросов
- Формирование отчетов
- (Доступно через главное окно администратора)
