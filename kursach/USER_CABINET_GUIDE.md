# 👤 РУКОВОДСТВО ПОЛЬЗОВАТЕЛЬСКОГО КАБИНЕТА

## 📋 Обзор функциональности

Личный кабинет пользователя предоставляет полный доступ к просмотру и анализу данных об энергопотреблении в компании. Все функции работают в режиме **только для чтения** (просмотр без редактирования).

---

## 🎯 Основные возможности

### 1️⃣ Вкладка "Здания"
Просмотр списка всех зданий организации.

**Доступная информация:**
- ID здания
- Название здания
- Адрес

**Функции:**
- 📊 Сортировка по любому столбцу (клик на заголовок)
- 📋 Полная таблица всех зданий

---

### 2️⃣ Вкладка "Помещения"
Просмотр помещений с фильтрацией по зданиям.

**Доступная информация:**
- ID помещения
- ID здания
- Название здания
- Этаж
- Категория помещения

**Фильтры:**
- 🏢 **Фильтр по зданию** - выберите здание из списка
- 🔄 **Показать все** - отобразить помещения всех зданий

**Как использовать:**
1. Откройте вкладку "Помещения"
2. Выберите здание из комбо-бокса "Фильтр по зданию"
3. Таблица автоматически обновится, показав помещения выбранного здания
4. Нажмите "Показать все", чтобы вернуться к полному списку

---

### 3️⃣ Вкладка "Энергопотребление"
Просмотр данных об энергопотреблении с многоуровневой фильтрацией.

**Доступная информация:**
- ID потребления
- ID помещения
- Дата
- Период времени
- Значение потребления (кВт)

**Фильтры:**
- 🏢 **Здание** - фильтр по зданию
- 🚪 **Помещение** - фильтр по помещению (обновляется при выборе здания)
- ⏰ **Период** - фильтр по времени суток

**Как использовать:**
1. Откройте вкладку "Энергопотребление"
2. (Опционально) Выберите здание - таблица обновится, и комбо "Помещение" заполнится помещениями из этого здания
3. (Опционально) Выберите конкретное помещение
4. (Опционально) Выберите период времени (например, "Ночь", "День", "Вечер")
5. Таблица обновится с учетом всех выбранных фильтров

**Примечание:** Фильтры работают в режиме И (AND), т.е. применяются все одновременно.

---

### 4️⃣ Вкладка "Поиск"
Расширенный поиск данных об энергопотреблении по сложным критериям.

**Критерии поиска:**
- 🏢 **Здание** - выберите здание
- 📅 **От даты** - начальная дата (включительно)
- 📅 **До даты** - конечная дата (включительно)

**Как использовать:**
1. Откройте вкладку "Поиск"
2. (Опционально) Выберите здание из комбо-бокса
3. (Опционально) Установите начальную дату, нажав на поле "От даты"
4. (Опционально) Установите конечную дату, нажав на поле "До даты"
5. Нажмите кнопку **"Искать"**
6. Результаты появятся в таблице ниже

**Примеры поиска:**
- Энергопотребление в определенном здании за месяц
- Все данные после конкретной даты
- Данные в диапазоне дат для всех зданий

---

### 5️⃣ Вкладка "Отчеты"
Аналитические отчеты и статистика энергопотребления.

**Типы отчетов:**

#### 📊 Общее энергопотребление по зданиям
Показывает статистику по зданиям:
- **Название здания** - наименование объекта
- **Общее потребление (кВт)** - суммарное потребление за весь период
- **Среднее потребление (кВт)** - среднее значение
- **Максимум (кВт)** - пиковое значение потребления
- **Количество записей** - число записей в расчете

#### 🚪 Энергопотребление по помещениям
Детальный список всех записей по помещениям:
- **Помещение** - ID и этаж помещения
- **Здание** - название здания
- **Потребление (кВт)** - значение
- **Дата** - дата измерения
- **Период** - время суток

#### ⏰ Энергопотребление по периодам
Агрегированная статистика по периодам времени:
- **Период времени** - название периода (Ночь, День и т.д.)
- **Общее потребление (кВт)** - суммарное потребление за период
- **Среднее потребление (кВт)** - среднее значение
- **Количество записей** - число записей

**Как сформировать отчет:**
1. Откройте вкладку "Отчеты"
2. Выберите тип отчета из комбо-бокса "Тип отчета"
3. Нажмите кнопку **"Сформировать отчет"**
4. Отчет появится в таблице ниже
5. Вы можете скопировать или экспортировать данные

---

## 🎨 Интерфейс и управление

### Панель управления
- **Заголовок** (синяя полоса вверху): Приветствие и кнопка "Выход"
- **Вкладки**: Переключение между разделами

### Таблицы данных (DataGrid)
- 📌 **Сортировка**: Клик на заголовок столбца для сортировки
- 🔄 **Скролл**: Используйте прокрутку для навигации
- 📋 **Копирование**: Выделите ячейки и скопируйте (Ctrl+C)

### Кнопки и элементы управления
- 🔘 **Комбо-боксы**: Нажмите, чтобы развернуть список
- 📅 **DatePicker**: Нажмите на поле для выбора даты из календаря
- 🔘 **Кнопки**: Нажмите для выполнения действия

## 🔒 Безопасность и ограничения

✅ **Все данные защищены**
- Вы видите только тот объем информации, на который у вас есть доступ
- Все операции находятся в режиме "только чтение"
- Удаление или редактирование невозможно

✅ **Сессия**
- Нажмите "Выход" для выхода из кабинета

## 📊 Примеры использования

### Пример 1: Как найти энергопотребление в конкретном здании за месяц

1. Откройте вкладку **"Энергопотребление"**
2. Выберите здание из списка "Здание"
3. Таблица обновится, показав только помещения этого здания
4. Выберите помещение (если нужно)
5. Выберите период времени (если нужно)

### Пример 2: Как сформировать отчет по статистике зданий

1. Откройте вкладку **"Отчеты"**
2. Выберите "Общее энергопотребление по зданиям"
3. Нажмите "Сформировать отчет"
4. Увидите таблицу со статистикой по каждому зданию

### Пример 3: Как искать данные за определенный период

1. Откройте вкладку **"Поиск"**
2. Нажмите на поле "От даты" и выберите начальную дату
3. Нажмите на поле "До даты" и выберите конечную дату
4. (Опционально) Выберите здание
5. Нажмите "Искать"
6. Результаты покажут все записи в выбранном диапазоне дат

## ⚙️ Технические детали

### Обновление данных
- Все таблицы обновляются в реальном времени при изменении фильтров
- Данные загружаются с центральной базы данных SQL Server

### Производительность
- Большие наборы данных могут требовать время на загрузку
- Используйте фильтры для более быстрого поиска

### Формат дат
- Формат даты: **ДД.ММ.ГГГГ** (например: 25.12.2024)
- Даты хранятся в UTC, отображаются в локальном времени

## 🆘 Решение проблем

### Проблема: Таблица пуста при выборе фильтра
**Решение:** Возможно, нет данных, соответствующих вашему фильтру. Попробуйте:
- Нажать "Показать все" в вкладке Помещения
- Выбрать другие фильтры
- Проверить дату в поиске

### Проблема: Данные загружаются долго
**Решение:** 
- Используйте более узкие фильтры (выберите одно здание)
- Ограничьте дату в поиске
- Подождите несколько секунд

### Проблема: Комбо-бокс "Помещение" пуст
**Решение:** Сначала выберите здание в комбо-боксе "Здание"
