## 🆘 Если не работает

**Проблема:** Файл не найден  
**Решение:** Убедитесь, что скопировали правильный путь: `C:\Users\danya\source\repos\kursach\kursach\Database_Setup.sql`

**Проблема:** "Сервер не найден"  
**Решение:** Проверьте, что SQL Server запущен: `Get-Service -Name MSSQLSERVER`

**Проблема:** "Доступ запрещен"  
**Решение:** Запустите PowerShell от администратора
